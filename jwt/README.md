### Installation 

- clonez le dépot `git clone https://github.com/guillaumesubra1/jwt.git`

### Lancement

- Ouvrez un `cmd` au dossier `jwt`
- Exécutez la commande `docker-compose up --build`
- Vérifiez que les conteneurs sont lancés avec la commande `docker ps`
- Une fois les 3 conteneurs lancés, ouvrez un navigeur et tapez `http://localhost:4200`
